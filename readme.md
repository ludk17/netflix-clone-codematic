## Instalar virtualenv
```
pip install virtualenv
```

## Configurar entorno virtual
```
virtualenv env
python -m virtualenv env
```

## Correr aplicación
```
python manage.py runserver
```

## Crear super usuario
```
python manage.py createsuperuser
```
